# Retro Creator — Documentation & Issues

Official public home of **Retro Creator**, the live overlay studio for retro-gaming
streams powered by real gameplay data (built on
[RetroBat APIExpose](https://github.com/Nelfe80/RetroBat-APIExpose)).

- 📚 **Documentation**: https://nelfe80.github.io/RetroCreator-Wiki/ (English & French)
- 🐛 **Official issue tracker**: [Issues](https://github.com/Nelfe80/RetroCreator-Wiki/issues)

Retro Creator is in active development and not released yet.
